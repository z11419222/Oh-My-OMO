## 💊 极致的 OMO 类固醇配置

这不是一个普通的编辑器。这是管理您 OMO “自律军团”的作战指挥台。我们把官方文档中每一个智能体的灵魂都注入到了界面中。

| 特性 | 功能说明 |
| :---: | :--- |
| 🗂️ | **具名预设管理 (Preset Hub)** | 瞬间切换任务模式。从“深度重构”切到“极速写文档”，只需一键。 |
| 🔌 | **提供商一键换源 (Provider Panel)** | Anthropic 的牢笼很漂亮，但我们不住那。快速切换 Kimi、OpenAI、DeepSeek 及本地源。 |
| 🎛️ | **高密度 UI 布局** | 专为 2K/4K 屏幕优化的三栏式紧凑布局。在单屏内掌控所有自律智能体的命运。 |
| 🛡️ | **安全性保障** | 每次应用配置前自动备份路由表。哈希级的修改准确度，绝不弄丢您的注释。 |

## 🤖 掌控您的自律军团 (Discipline Agents)

Oh My OMO 已深度集成所有官方智能体的职责定义，为您每一个职能类别自动匹配最强模型：

- **Sisyphus** (`主指挥官`): 制定计划、分配任务、全速推进。
- **Hephaestus** (`独立大匠`): 自主探索模式，独立完成闭环，绝不当保姆。
- **Prometheus** (`战略规划`): 动手前先问诊，用详尽计划替代无脑 Prompt。
- **Oracle** (`架构顾问`): 您深度调试和架构咨询的智囊团。


## 🛠️ 核心操作逻辑

我们摒弃了传统的“边改边存”逻辑，采用了更符合工程师直觉的 **“预设 -> 应用”** 流程：

1. **加载预设**：从下拉框中选择一个场景预设。
2. **离线调优**：自由修改模型、参数，界面会实时反馈变更，但不会影响当前运行。
3. **应用配置 (Apply)**：确认无误后点击 `应用配置`。配置将瞬间同步至 `.config/opencode/oh-my-opencode.jsonc`。

## 🎨 设计哲学

读代码文件的时代已经过去了，但手写 JSONC 的痛点依然存在。

Oh My OMO 的目的不是隐藏配置的复杂性，而是**通过高密度的呈现，赋予人类更强的管理能力**。我们保留了 JSONC 的每一条注释，但在展现形式上，我们选择了极致的紧凑与专业感。
